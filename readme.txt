Creates content blocks to insert into anywhere of a WooCommerce site! At 100+ places!

== Description ==
This is an add-on popular Woocommerce plugin.

It helps you create content blocks to insert into anywhere of a WooCommerce site!

You can create as many content blocks as you want and add them any of the 100+ places.

== Installation ==

1. Upload the `insert-into-woocommerce` folder and its contents to your `wp-contents/plugins` directory.
2. Activate in the `Plugins` menu.
3. Create content blocks from admin menu 'WooBlocks', choose location from the list and publish.
4. This block will be shown at selected location.
5. You are done!
